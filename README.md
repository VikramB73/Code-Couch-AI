# Coding Learning Platform

A platform for learning coding by analyzing problems and providing three different solutions: brute force, better, and optimal approaches.

## Setup

1. Install dependencies for both client and server.

2. Set up Gemini API key: Create a .env file in server/ with GEMINI_API_KEY=your_key

3. Start the server: cd server && npm start

4. Start the client: cd client && npm start

## Features

- Input a coding problem
- Get 3 solutions with code, complexities, and analysis
- Toggle detailed analysis for each approach
